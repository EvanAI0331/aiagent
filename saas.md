# 传统 SaaS 垂直行业服务列表

## 1. 通用企业经营类 SaaS

| 类别 | 典型服务 |
|---|---|
| CRM 客户关系管理 | 销售线索、客户管理、商机管理、销售漏斗、客户跟进 |
| ERP 企业资源计划 | 财务、采购、库存、生产、订单、供应链 |
| HRM 人力资源管理 | 招聘、入职、考勤、薪酬、绩效、员工档案 |
| OA 协同办公 | 审批流、公告、会议、流程管理、内部协作 |
| 财务 SaaS | 记账、报销、发票、预算、税务、财务报表 |
| 项目管理 | 任务管理、甘特图、项目进度、资源分配、交付管理 |
| 文档协作 | 在线文档、知识库、权限管理、版本管理 |
| 合同管理 | 合同起草、审批、签署、归档、续约提醒 |
| 电子签名 | 签署流程、身份认证、合同存证 |
| BI 数据分析 | 数据看板、经营分析、指标监控、报表自动化 |
| 低代码 / 无代码 | 表单、流程、内部系统搭建、自动化工具 |
| ITSM | 工单、资产管理、权限管理、故障处理 |
| RPA 自动化 | 重复流程自动执行、系统间数据搬运 |
| 客服系统 | 在线客服、工单、知识库、呼叫中心、机器人客服 |
| 营销自动化 | 邮件营销、短信营销、私域运营、用户分层 |
| CDP 用户数据平台 | 用户画像、行为分析、标签体系、营销触达 |
| 数据安全 | 权限控制、数据脱敏、审计、备份、合规 |
